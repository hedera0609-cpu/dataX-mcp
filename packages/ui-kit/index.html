11. DataXDB — データベースSDK

ℹ localhost → localStorage を自動使用 / 本番環境 → DynamoDB へ自動切替。コード修正不要。
初期化
const db = new DataXDB(); const todos = db.collection('todos');
Create
const id = await todos.add({ text: 'タスクを完了する', done: false });
Read
// 全件取得 const all = await todos.get(); // フィルタ取得 const done = await todos.get({ done: true }); // 1件取得 const item = await todos.doc(id).get();
Update / Delete
await todos.doc(id).update({ done: true }); await todos.doc(id).delete();

15. ❌ 禁止事項（AI注意）

⚠ 以下のパターンを使用すると、ダーク/ライトモードが崩れたり、アプリ間のデザイン一貫性が失われます。
❌ インラインスタイルで色・ボタンを直接指定する <button style="background:blue;color:white;padding:8px 16px">NG</button> ❌ 独自CSSクラスでdx-スタイルを上書きする <style>.my-button { background: red; }</style> ❌ CSS変数を直接上書きする <style>:root { --dx-bg: pink; }</style> ✅ 正しい方法：dx-クラスを使用する <button class="dx-btn dx-btn-primary">OK</button> ✅ カスタムデザインが必要な場合：dx-変数を参照する <style> .my-component { background: var(--dx-bg-secondary); border: 1px solid var(--dx-border); border-radius: var(--dx-radius); } </style>
